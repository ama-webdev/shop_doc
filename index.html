Installation

  • # git clone project
  • # composer install
  • # key generate
  • # start server

Git clone porject

ပထမဆုံး project ကို clone ဖို့အတွက် htdocs folder ထဲမှာ right click နှိပ် open in terminal ကိုရွေးပြီး အောက်က command ကို run ပေးပါ၊ 

                    
                        git-clone https://github.com/ama-webdev/shop.git

Composer install

ပြီးရင် project မှာပါမယ့် package တွေကို composer အကူအညီနဲ့ install လုပ်ပါမယ်။ install လုပ် ဖို့အတွက် project folder ထဲမှာ terminal ဖွင့်ပြီး အောက်က command ကို ပေးပါ။ 
                    
                        composer install

Key generate

laravel project တစ်ခု run ဖို့အတွက် key တစ်ခု လိုပါတယ်။ project ကို create လုပ်ရင် key ကို auto generate လုပ်ပေးပေမယ့် git က clone ထားရင်တော့ generate လုပ်ပေးဖို့ လိုပါတယ်။ အောက်က command ကို run ပေးပါ။ 

                    
                        php artisan key:generate

start server

                    
                        php artisan serve
